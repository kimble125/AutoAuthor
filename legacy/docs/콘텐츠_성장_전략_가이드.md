## 1. AutoAuthor 서비스 개요

### 1-1. 서비스의 궁극적 목적

AutoAuthor의 핵심 목적은 **작품 평가 자체가 아니라, 내 블로그 글·유튜브 영상 콘텐츠의 생산과 그 조회수·체류시간 극대화**입니다. 이를 위해 콘텐츠 기획·생성·업로드를 자동화하며, 기획 단계는 최소 3단계로 구성됩니다.

```
[기획 1단계] 화제성 탐지
현재 많이 검색되는 작품 목록 수집 → 화제성 등급 부여

[기획 2단계] 포화도 측정
해당 작품이 이미 얼마나 많은 콘텐츠로 만들어졌는지 측정 → 포화도 등급 부여

[기획 3단계] 키워드 기획안 생성
화제성 높고 포화도 낮은 작품의 SEO 키워드 탐색 → 제목·소제목·콘텐츠 기획안 자동 생성
```

### 1-2. '포화도'라는 용어에 대한 피드백

> **(질문 답변)** 기존에 사용하던 '경쟁도'보다 **'포화도(飽和度)'가 더 적합한 용어**입니다.

'경쟁도'는 내가 싸워야 할 상대방의 강도에 초점을 맞추는 반면, '포화도'는 **시장이 얼마나 채워져 있는가**라는 상태를 표현합니다. 크리에이터 입장에서 중요한 것은 "내가 이길 수 있냐"가 아니라 "이 시장에 빈자리가 있냐"이므로, 포화도가 서비스의 본질적 가치를 더 정확하게 전달합니다.

또한 '블루 오션 / 레드 오션' 같은 바다 비유와 자연스럽게 연결되어 직관적인 UI/UX 설계에도 유리합니다.

## 2. 핵심 지표 시스템: 화제성 × 포화도 4단계

> 이 시스템은 **로튼 토마토의 '신선도 지수'**, **CGV의 '깨진 달걀'** 방식을 참고하여, 크리에이터에게 즉각적인 경각심과 기대감을 동시에 주는 직관적 UI/UX로 설계되었습니다.

---

### 2-1. 지표 A: 화제성 📈 (대중의 수요 / 검색량)

*"유저들이 이 작품을 얼마나 미친 듯이 검색하고 있는가?"*

| 등급 | 아이콘 | 명칭 | 상태 | 크리에이터 시그널 |
|---|---|---|---|---|
| 4 | 👑 | **천상계 도달** | 검색량 폭발. 시장을 지배한 넘사벽 1위. | "지금 대한민국에서 이 작품 모르면 간첩입니다." |
| 3 | 📈 | **우상향 탑승** | 입소문을 타기 시작해 검색량 그래프가 가파르게 치솟는 중. | "트렌드에 가장 민감하게 반응하는 지금이 탑승 적기입니다." |
| 2 | 💬 | **잔잔한 수요** | 폭발적이진 않지만 충성도 높은 매니아층이 꾸준히 검색하는 상태. | "롱테일(장기) 유입을 노리기 좋은 안정적인 국밥 키워드입니다." |
| 1 | 🧊 | **냉동 창고** | 대중의 관심이 차갑게 식어 검색량이 거의 없는 상태. | "아무도 찾지 않습니다. 굳이 꺼내보지 마세요." |

**점수 기준 (구글 트렌드 KR 기준):**

| 점수 범위 | 등급 |
|---|---|
| 75~100 | 👑 천상계 도달 |
| 45~74 | 📈 우상향 탑승 |
| 15~44 | 💬 잔잔한 수요 |
| 0~14 | 🧊 냉동 창고 |

### 2-2. 지표 B: 포화도 🌊 (타 크리에이터의 공급량)

*"내가 글을 썼을 때 상위 노출(1페이지)을 먹을 수 있는가?"*

| 등급 | 아이콘 | 명칭 | 상태 | 크리에이터 시그널 |
|---|---|---|---|---|
| 1 | 🫧 | **블루 오션** | 오염되지 않은 청정 1급수 구역. | "발자국 하나 없는 청정 구역. 대충 써도 상위 노출 확정!" |
| 2 | ⛵ | **퍼플 오션** | 나만의 돛을 달고 항해하기 좋은 틈새 바다. | "경쟁자가 약간 있지만, 독창적인 기획으로 충분히 뚫을 수 있는 기회의 땅." |
| 3 | 🦈 | **레드 오션** | 상어 떼가 우글거리며 파이를 뜯어 먹는 피 튀기는 바다. | "이미 수많은 크리에이터가 달려든 포화 상태. 웬만한 블로그 체급이 아니면 살아남기 힘듭니다." |
| 4 | ☠️ | **블랙 오션** | 빛도 들어오지 않는 심해이자 무덤. | "대형 언론사·나무위키·탑급 인플루언서들이 꽉 잡고 있는 데드 존. 당장 도망치세요." |

**점수 기준 (네이버 블로그 검색 결과 수 기준):**

| 경쟁 포스트 수 | 등급 |
|---|---|
| 0~5개 | 🫧 블루 오션 |
| 6~20개 | ⛵ 퍼플 오션 |
| 21~100개 | 🦈 레드 오션 |
| 100개 초과 | ☠️ 블랙 오션 |

유저가 AutoAuthor에서 작품을 검색하면 아래와 같은 명쾌한 결과 창을 보게 됩니다.

**📋 AutoAuthor 분석 리포트: 넷플릭스 《레이디 두아》**

- **🤖 AI 기획 코멘트:** "청정 구역입니다! 경쟁자들이 몰려오기 전에 당장 '캐릭터 비평' 주제로 기획안을 생성하세요."

**📋 AutoAuthor 분석 리포트: 영화 《파묘》**

## 3. V3 코드 사용법

### 3-1. 최초 1회 환경 설정

Windsurf 터미널(하단 Terminal 탭)을 열고 아래 명령어를 순서대로 실행합니다.

```bash
# 필요한 파이썬 패키지 설치 (최초 1회만)
pip install requests beautifulsoup4 google-generativeai
```

설치 완료 후 `Successfully installed` 메시지가 뜨면 정상입니다.

### 3-2. Gemini API 키 발급 (블로그 제목 자동 생성 기능)

1. [aistudio.google.com](https://aistudio.google.com) 접속 → Google 계정으로 로그인
2. 좌측 메뉴 **Get API key** 클릭 → **Create API key** 버튼 클릭
3. 키 복사 (화면을 벗어나도 재확인 가능, 분실 시 새로 발급)
4. `mvforrest_seo_v3.py` 파일 상단 CONFIG 섹션에 붙여넣기:

```python
GEMINI_API_KEY = "AIza여기에붙여넣기"
```

### 3-3. 모델 선택 (선택 사항)

| 모델명 | 한국어 품질 | 속도 | 비용 | 추천 상황 |
|---|---|---|---|---|
| `gemini-2.5-pro` | ★★★★★ | 보통 | 중간 | **기본값, 최고 품질 필요 시** |
| `gemini-2.5-flash` | ★★★★☆ | 빠름 | 저렴 | 빠른 초안이 필요할 때 |
| `gemini-2.5-flash-lite` | ★★★☆☆ | 매우 빠름 | 매우 저렴 | 대량 처리 시 |

> **비용 안내:** Gemini 2.5 Pro 기준 1회 분석(키워드 50개 + 제목 생성)에 약 **$0.001~0.005** (약 1~7원) 소요됩니다.

### 3-4. 분석할 키워드 설정

```python
# 리뷰 쓸 영화/드라마 제목을 여기에 입력하세요
SEED_KEYWORDS = [
    "영화제목",           # 예: "베테랑2"
    "감독이름 영화",      # 예: "류승완 감독"
    "OTT 영화제목",       # 예: "넷플릭스 베테랑2"
]
```

### 3-5. 코드 실행 및 결과 확인

```bash
cd 코드파일이있는폴더경로
python mvforrest_seo_v3.py
```

| 생성 파일 | 내용 |
|---|---|
| `키워드분석_일반_날짜시간.csv` | 전체 키워드 점수표 (엑셀로 열기 가능) |
| `황금키워드_일반_날짜시간.txt` | 황금 키워드 목록 + Gemini AI 블로그 제목 초안 |

## 4. 분석 결과 해석 방법

### 4-1. CSV 컬럼 설명

| 컬럼 | 의미 | 해석 방법 |
|---|---|---|
| `키워드` | 분석된 키워드 | — |
| `수요(구글자동완성)` | O/X | O = 실제 검색 수요 있음 |
| `블로그경쟁도` | 0~10+ | 낮을수록 포화도 낮음 |
| `VIEW경쟁도` | 0~10+ | 낮을수록 포화도 낮음 |
| `기회점수` | 0~100 | **높을수록 블루오션** |
| `포화도등급` | 블루/퍼플/레드/블랙 오션 | 블루·퍼플 오션 우선 공략 |
| `검색의도` | 정보형/탐색형/거래형 | 정보형이 블로그에 가장 적합 |

### 4-2. 포화도 등급별 활용 전략

**🫧 블루 오션 (기회점수 50점 이상):** 블로그 제목의 앞부분에 반드시 배치합니다. 본문 첫 문단, 주요 소제목(H2), 이미지 alt 텍스트에 자연스럽게 포함시킵니다.

**⛵ 퍼플 오션 (20~49점):** 블루 오션 키워드와 함께 제목에 조합하거나, 본문 소제목(H3)과 태그로 활용합니다.

**🦈 레드 오션 (10~19점):** 제목에는 사용하지 않되, 본문 내 자연스러운 언급은 괜찮습니다.

**☠️ 블랙 오션 (10점 미만):** 해당 키워드 자체는 포기하고, AI 코멘트에서 제안하는 세부 우회 키워드를 활용합니다.

## 5. 티스토리 블로그 SEO 최적화 전략

### 5-1. MVforrest 블로그 현황 진단

블로그를 직접 분석한 결과, 글 품질과 분석 깊이는 상위 5% 수준입니다. 그러나 SEO 구조 측면에서 아래 개선 여지가 있습니다.

| 항목 | 현재 상태 | 개선 방향 |
|---|---|---|
| 카테고리 구조 | 분류 전체보기 41개 (단일 카테고리) | 장르별 카테고리 세분화 |
| 제목 구조 | 키워드 포함, 다소 길이 가변적 | 블루 오션 키워드를 앞부분에 고정 배치 |
| 시리즈 연재 | ①② 형식 활용 중 | 내부 링크 연결 강화 |
| 이미지 최적화 | 확인 불가 | alt 텍스트에 키워드 포함 필수 |
| 발행 주기 | 주 2~3회 | 주 3~4회로 증가 목표 |

### 5-2. 제목 작성 공식

```
[블루/퍼플 오션 키워드] + 《작품명》 + [차별화 포인트] + [독자 이익]

예시:
"베를린 휴민트 세계관 완벽 정리 | 류승완 첩보 유니버스 연결고리 총정리"
"휴민트 실화 배경 분석 | 블라디보스토크 국경 정보전의 실제 역사"
```

### 5-3. 본문 SEO 체크리스트

| 체크 항목 | 방법 |
|---|---|
| 블루/퍼플 오션 키워드를 제목 앞 20자 이내에 배치 | 제목 직접 수정 |
| 첫 문단(100자 이내)에 핵심 키워드 포함 | 도입부 작성 시 의식적으로 포함 |
| 소제목(H2) 2~3개에 퍼플 오션 키워드 배치 | 소제목 작성 시 키워드 자연스럽게 삽입 |
| 대표 이미지 alt 텍스트에 키워드 포함 | 이미지 업로드 시 alt 텍스트 입력 |
| 태그 5~10개 설정 (블루+퍼플 오션 키워드 조합) | 발행 전 태그 입력 |
| 이전/다음 글 내부 링크 연결 | 본문 하단에 관련 글 링크 추가 |

## 6. 크로스 키워드 전략 — 여러 작품·이슈 조합

### 6-1. 크로스 키워드란 무엇인가

단일 작품 리뷰 키워드는 개봉 직후 검색량이 급등했다가 빠르게 소멸합니다. 반면 **두 개 이상의 작품이나 사회적 이슈를 연결하는 크로스 키워드**는 관련 사건이 재발하거나 유사 작품이 개봉할 때마다 검색량이 다시 올라오는 **상록 콘텐츠(Evergreen Content)**가 됩니다.

> **예시:** "계엄 선포 후 다시 보는 영화 1987"이라는 글은 2024년 12월 계엄 사태 당시 검색량이 급등했고, 이후 관련 정치적 사건이 발생할 때마다 반복적으로 유입이 발생합니다.

### 6-2. 크로스 키워드의 3가지 유형

| 유형 | 설명 | 예시 |
|---|---|---|
| **이슈 × 작품** | 현재 사회 이슈와 관련 영화·드라마 연결 | "계엄 영화 1987", "내란 서울의 봄" |
| **작품 × 작품** | 유사한 주제를 다룬 두 작품 비교 | "베를린 vs 휴민트 첩보 영화 비교" |
| **장르 × 시대** | 특정 장르의 역사적 흐름 분석 | "한국 첩보 영화 계보 베를린 휴민트" |

### 6-3. 크로스 키워드 모드 사용법

```python
# 크로스 모드 활성화
CROSS_MODE = True

# 연결할 주제 쌍 입력
CROSS_TOPICS = [
    ("계엄",   "영화 1987"),
    ("계엄",   "그것이 알고싶다"),
    ("내란",   "서울의 봄"),
    ("민주화", "택시운전사"),
]
```

### 6-4. 크로스 콘텐츠 제목 작성 공식

```
[시의성 이슈 키워드] + [작품명 A] + [연결 포인트] + [독자 이익]

예시:
"계엄 선포 후 다시 보는 영화 1987 | 그것이 알고싶다가 짚은 역사의 반복"
"내란과 계엄 다룬 한국 영화 추천 5편 | 1987부터 서울의 봄까지 현실이 된 스크린"
"베를린 vs 휴민트 비교 분석 | 류승완 첩보 유니버스의 진화"
```

### 6-5. 크로스 콘텐츠 발행 타이밍 전략

| 발행 시점 | 효과 | 전략 |
|---|---|---|
| 이슈 발생 후 24시간 이내 | 검색량 급등 구간 선점 | 크로스 모드로 즉시 키워드 분석 후 빠르게 발행 |
| 이슈 발생 후 1~2주 | 심층 분석 수요 증가 | 배경·역사·맥락 분석 글로 차별화 |
| 이슈 소강 후 | 상록 콘텐츠로 전환 | 내부 링크 연결해 관련 글 클러스터 구성 |

## 7. 멀티채널 통합 전략

### 7-1. 원소스 멀티유즈(OSMU) 구조 설계

```
[블로그 심층 리뷰 1편 작성]
        ↓
┌───────────────────────────────────────────┐
│  유튜브: 5~10분 영상 리뷰 (블로그 핵심 요약)  │
│  인스타: 카드뉴스 3~5장 (핵심 포인트 시각화)  │
│  네이버: 블로그 요약본 (SEO 별도 최적화)      │
│  스레드/X: 핵심 문장 1~3개 발췌 공유         │
└───────────────────────────────────────────┘
```

### 7-2. 채널별 역할 분담

| 채널 | 주요 역할 | 콘텐츠 형식 | 수익화 방식 |
|---|---|---|---|
| **티스토리** | SEO 트래픽 허브 | 2,000~4,000자 심층 리뷰 | 카카오 애드핏 |
| **유튜브** | 영상 광고 수익 + 구독자 확보 | 5~10분 리뷰 영상 | 유튜브 광고 + 슈퍼챗 |
| **인스타그램** | 브랜딩 + 팔로워 유입 | 카드뉴스, 릴스 | 협찬, 제휴 마케팅 |
| **네이버 블로그** | 네이버 검색 트래픽 확보 | 요약본 + 티스토리 링크 | 네이버 애드포스트 |

### 7-3. 인스타그램 화제성 등급 활용 전략

AutoAuthor의 화제성·포화도 등급 아이콘을 인스타그램 스토리·릴스에 직접 활용합니다.

- **스토리 활용:** "이번 주 👑 천상계 도달 작품은?" → 블로그 링크 유도
- **릴스 활용:** "🫧 블루 오션 발견! 지금 당장 봐야 할 이유" → 호기심 자극
- **카드뉴스 활용:** 화제성 × 포화도 매트릭스를 인포그래픽으로 시각화

## 8. 수익화 로드맵

### 8-1. 단계별 수익화 전략

**Phase 1 (0~6개월): 트래픽 기반 구축**

이 단계에서는 수익보다 콘텐츠 자산 축적에 집중합니다. 티스토리 카카오 애드핏을 설정해두되, 월 수익은 미미할 수 있습니다. 목표는 월 방문자 5,000명 이상 달성입니다.

**Phase 2 (6~12개월): 수익 채널 다각화**

유튜브 파트너 프로그램(YPP) 가입 조건 달성 후 광고 수익 시작. 네이버 블로그 애드포스트 병행. 인스타그램 팔로워 1만 명 목표로 협찬 문의 대응 준비.

**Phase 3 (12개월 이후): 고수익 수익화**

제휴 마케팅(OTT 구독 링크, 영화 예매 링크), 유료 뉴스레터, AutoAuthor 서비스 상품화(SaaS) 등 직접 수익화 채널 추가.

### 8-2. 수익화 수단별 예상 수익

| 수익화 수단 | 조건 | 예상 월 수익 (안정기) |

| 유튜브 광고 | 구독자 1만명, 월 조회수 10만 | 20~80만원 |
| 네이버 애드포스트 | 월 방문자 5,000명 | 2~5만원 |
| 인스타 협찬 | 팔로워 1만명 | 10~50만원/건 |
| OTT 제휴 마케팅 | 링크 클릭 기반 | 5~30만원 |

## 10. 콘텐츠 제작 주간 루틴

SEO 분석부터 발행까지의 표준 워크플로우입니다. 이 루틴을 따르면 **콘텐츠 1편 제작에 3~4시간**으로 최적화할 수 있습니다.

### 월요일: 화제성·포화도 탐지 (30분)

```bash
# 이번 주 리뷰할 영화/드라마 제목을 SEED_KEYWORDS에 입력 후 실행
python mvforrest_seo_v3.py
```

`황금키워드_날짜시간.txt` 파일을 열어 AI가 생성한 제목 초안 5개를 확인합니다. 화제성 등급과 포화도 등급을 교차하여 이번 주 공략 작품을 최종 선택합니다.

### 화~목요일: 콘텐츠 제작 (2~3시간/편)

블루/퍼플 오션 키워드를 제목 앞부분에 배치하고, 본문 소제목에 자연스럽게 포함합니다. 발행 전 SEO 체크리스트(5-3항 참조)를 반드시 확인합니다.

### 금요일: 멀티채널 배포 (1시간)

블로그 발행 후 인스타그램 카드뉴스 3장 제작(Canva), 네이버 블로그 요약본 발행, 스레드/X 핵심 문장 공유 순으로 진행합니다.

### 주말: 성과 분석 (30분)

티스토리 통계에서 조회수 상위 글을 확인하고, 다음 주 키워드 방향에 반영합니다.
